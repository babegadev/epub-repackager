# EPUB Web Fixer

A browser-only EPUB repair tool for fixing Apple Books exploded folders and broken EPUB archives.

## Development

1. Install dependencies with `npm install`.
2. Start the dev server with `npm run dev`.
3. Build with `npm run build`.

## Features

- Upload multiple EPUB files at once.
- Upload exploded folders using the folder picker.
- Repair files locally in the browser.
- Download items individually or as a bulk zip.
